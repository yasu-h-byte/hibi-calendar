# HIBI CONSTRUCTION 就業カレンダーシステム - 開発ルール

## 会社概要
- 会社名：株式会社日比建設（東京都清瀬市）
- 事業：とび・土工工事
- 従業員：日本人社員8名（日給月給制）、特定技能7名・技能実習生3名（月給制、全員ベトナム人）
- 監理団体：エムテック協同組合（担当：Chiさん）

## 労働時間のルール
- 所定労働時間：1日7時間
- 始業：8:00 / 終業：17:00
- 休憩：10:00〜10:30（30分）+ 12:00〜13:00（60分）+ 15:00〜15:30（30分）= 合計2時間
- 月間所定労働時間：168時間
- 残業：月168時間を超えた分のみ割増賃金（25%）

## 変形労働時間制のルール（1ヶ月単位）
- 毎月20〜25日頃に元請から翌月カレンダーを入手
- 翌月末日までに就業カレンダーを確定・交付
- 従業員への署名取得が必須（このシステムで実施）

## カレンダーパターン定義
- パターンA：土曜フル稼働・祝日休み（月間所定日数25〜26日）
- パターンB：土曜隔週（第2・4土曜休み）・祝日休み（月間所定日数23〜24日）
- パターンC：土曜全休・祝日休み（月間所定日数21〜22日）

## 色分けルール（スタッフ画面）
- 稼働日：青 / đi làm（出勤）
- 休日（土日）：グレー / nghỉ（休み）
- 祝日：赤 / nghỉ lễ（祝日）
- 有給：緑 / nghỉ phép（有給休暇）

## 言語表示ルール
- 管理者画面：日本語のみ
- スタッフ画面：日本語とベトナム語を必ず並記
- 署名ボタン：「内容を確認しました / Tôi đã xác nhận nội dung」

## 署名のルール
- 署名後はデータをロックして変更不可
- 署名記録にはタイムスタンプを保存
- 未署名者は管理画面で赤く表示

## Firestoreのデータ参照
- 既存の出面管理システムのデータを共有
- WORKERSデータ：demmen コレクション → main ドキュメント → workers配列
- トークン認証：workersデータの token フィールドを使用

## デザイン
- メインカラー：#1B2A4A（HIBI ネイビー）
- スタッフ画面：スマホ最適化・フォント16px基本・タップターゲット44px以上
- 管理者画面：PCレイアウト優先
