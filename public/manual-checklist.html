HIBI CONSTRUCTION 就業管理システム
最終更新: 2026年7月3日

このページの使い方：1ヶ月の業務の流れを「いつ・誰が・何をするか」で時系列に把握するための入口ページです。
まず下の月次タイムラインで全体の流れをつかみ、詳しい操作方法は各ステップからリンクされている担当マニュアルを開いてください。

📅 月次タイムライン

就業カレンダーの作成から給与計算・月締めまで、1ヶ月がどう回るかを時系列で示します。色は担当ロール（🔴管理者 / 🟠政仁さん / 🔵職長 / 🟣事務 / 🟢スタッフ / ⚙️自動）を表します。

━ 前月：翌月の準備 ━
前月 〜25日
🔵 職長
翌月の就業カレンダーを作成・提出
担当現場の翌月稼働日を設定し、政仁さんに提出する。25日までが期限。
→ 職長マニュアル
前月 25日〜末
🟠 政仁さん
翌月カレンダーを承認
職長が提出した翌月の就業カレンダーを確認し、最終承認する。
→ 政仁さんマニュアル
前月末〜当月初
🟢 スタッフ
翌月の就業カレンダーに署名
承認済みの翌月予定が届いたら、スマホで内容を確認して署名する。
→ スタッフ向けマニュアル
━ 当月：毎日まわる出面サイクル ━
当月 毎日
🟢 スタッフ
スマホで出勤入力
その日の始業・終業時刻と休憩を入力。休む時は有給申請・欠勤届も同じ画面から。
→ スタッフ向けマニュアル
当月 毎日
🔵 職長
担当現場の出面を確認・職長承認（ロック）
スタッフの入力を確認し、誤りを修正して職長承認。休暇申請は職長が第1段階の承認をする。
→ 職長マニュアル
当月 毎日
🟠 政仁さん
出面と休暇申請を最終承認
職長承認済みの出面を最終承認（ロック）。有給・帰国申請も政仁さんが最終承認する。
→ 政仁さんマニュアル
当月 随時
🟣 事務（奥寺さん）
入力補助・休暇アラート対応
スタッフが入力できなかった分の代理入力・過去日修正。「年5日未達」「繰越時効間近」等に対応。
→ 奥寺さんマニュアル
━ 翌月初：締めと給与計算 ━
翌月 1日〜10日

月次集計・給与計算・帳票出力
前月の出面を月次集計画面で確認し、給与額を確定。出面表・給与明細・有給台帳をExcel出力。

翌月 10日ごろ

👥 ロール別クイックガイド

各ロールが「毎日・毎月・年次」に何をするかの早見表です。詳しい手順は各カードの担当マニュアルへ。

🔴 管理者（靖仁さん）
システム設定・スーパー操作・有給付与の運用
  • 毎日ダッシュボードで全体状況を確認、通知ベルでアラート対応
  • 毎月有給の半自動付与（バナー確認→ワンクリック）／時効処理は自動
  • 年次日本人の新FY付与・9月末買取記録、有給管理簿Excel出力（労基監査）、退職・採用の設定変更
→ 休暇管理マニュアル
🟠 政仁さん（事業責任者）
最終承認・原価管理・経営判断
  • 毎日稼働状況の把握、出面の最終承認、休暇申請の最終承認（勤怠申請カードで対応待ちを確認）
  • 毎月翌月カレンダーの承認、月締め後の原価・収益の確認
  • 年次評価の最終承認・昇給決定 （評価制度の運用開始後）
→ 政仁さんマニュアル
🔵 職長（現場責任者）
現場運営・出面の第1段階承認・カレンダー作成
  • 毎日自分の出勤入力、担当現場の出面を確認・職長承認
  • 随時休暇申請の職長承認（第1段階。最終承認は政仁さん）
  • 毎月翌月カレンダーを25日までに作成・提出
  • 年次担当スタッフの評価入力（第1段階）（評価制度の運用開始後）
→ 職長マニュアル
🟣 事務（奥寺さん）
給与計算・帳票出力・各種データ管理
  • 毎日出面入力の補助・修正、休暇アラート対応、試験日は「試」入力
  • 月次月次集計・給与計算（翌月1〜10日）、帳票Excel出力、月締めロック
→ 奥寺さんマニュアル／給与計算マニュアル
🟣 事務（佐藤さん・道具代担当）
道具代補助の管理
  • 随時領収書を受け取り、道具代補助の購入記録を登録
  • 年次10月に年度予算3万円を新しい期間へ切り替え
→ 佐藤さんマニュアル
🟢 スタッフ（実習生・特定技能）
自己出勤入力・各種申請
  • 毎日スマホで始業・終業・休憩を入力
  • 随時有給申請（事前）・欠勤届・帰国申請（3ヶ月前）。承認前なら取り消し可
  • 毎月翌月カレンダーの署名、有給・道具代の残数確認
→ スタッフ向けマニュアル（ベトナム語）
