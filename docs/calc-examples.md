## 例1: 標準月（20日勤務・残業なし・補償なし）

**条件:**
- 雇用区分: 時給制ベトナム人（新ルール）
- 時給: ¥1,500、ベース日数: 20、月: 31日（5月）
- 出勤: 20日（各日7h所定通り）
- 有給: 0日、補償: 0日、欠勤: 0日
- 残業: 0h、深夜: 0h、法定休日労働: 0h

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 1,500 × max(0, 20-20) × 7 | 0 |
| 所定外労働手当 | 0h × 1,500 | 0 |
| 法定外残業手当 | 0h × 1,500 × 1.25 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | | **210,000** |

## 例2: 残業発生月（出勤18日 + 有給2日 + 残業18.5h）

**条件:** 例1と同じ条件で、出勤18日、有給2日、残業合計18.5h（うち3層判定 statutoryOT = 1.0h）

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 1,500 × max(0, 18-20) × 7 | 0 |
| 有給日給 | 出勤18+有給2=20日 ≤ 20日枠 → 0日 | 0 |
| **所定外労働手当** | **1,500 × 1.0 × 18.5h（所定超の全部）** | **27,750** |
| **法定外残業手当** | **1,500 × 0.25 × 1.0h（割増分のみ）** | **375** |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | max(0, 20 - 18 - 2 - 0) = 0 → -0 | 0 |
| **支給額** | | **238,125** |

**ポイント:** 残業18.5hは全時間を所定外労働手当（1.0倍）で支払い、3層判定で法定外残業に該当する
1.0h には**割増分0.25倍のみ**を法定外残業手当として追加する（1.0 + 0.25 = 実効1.25倍。労基法24条・37条）。
合計は旧分解（26,250 + 1,875）と同額だが、項目別内訳はこの形で出力される。

## 例3: 補償日あり（出勤15日 + 補償5日）

**条件:** 例1と同じ条件で、出勤15日、補償（w=0.6）5日

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 0 | 0 |
| 所定外労働手当 | 0 | 0 |
| 法定外残業手当 | 0 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| **休業手当** | **1,500 × 7 × 0.6 × 5** | **31,500** |
| **欠勤控除** | max(0, 20-15-0-0) = 5日 → **-1,500 × 7 × 5** | **-52,500** |
| **支給額** | 210,000 + 31,500 - 52,500 | **189,000** |

**ポイント:** 補償日（休業日）は基本給から1日分を控除（欠勤扱い）、別途 60% を休業手当で支給。
**正味で 60% 支給**（労基法26条準拠）。

**ポイント:** 法定休日（日曜）労働は 8h以下 × 1.35 + 8h超 × 1.60 で算定（労基法37条）。

## 例5: 月60h超残業（出勤22日・残業70h）

**条件:** 例1と同じ条件で、出勤22日、残業合計70h（うち statutoryOT = 70h）

| 項目 | 計算 | 金額 |
|---|---|---|

| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | | **366,000** |

**ポイント:** 残業70hは全時間を所定外労働手当（1.0倍）で支払い、法定外残業手当は割増分のみ
（60h以下 +0.25、60h超 +0.5。実効 1.25倍／1.5倍。労基法37条1項ただし書、2023/4 全企業適用）。

## 例7: 中途入社月（6/15入社、6月分）

**条件:** 6/15入社（在籍 16/30 暦日 = 53%）、出勤10日（うち全て規定通り）、残業なし

| 項目 | 計算 | 金額 |
|---|---|---|
| 按分後 baseDays | round(20 × 16/30) | 11 |
| 基本給（固定） | 1,500 × 11 × 7 | 115,500 |
| 追加所定 | max(0, 10-11) × 7 × 1,500 | 0 |
| 所定外労働手当 | 0 | 0 |
| 法定外残業手当 | 0 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | max(0, 11-10-0-0) = 1日 → -1 × 7 × 1,500 | -10,500 |
| **支給額** | 115,500 - 10,500 | **105,000** |

**ポイント:** 在籍期間で baseDays を按分（暦日数比例）。
中途入社月の過大な欠勤控除を防ぐ。
固定月給者（完全月給・フン）も同様に基本給を日割り（2026-06-12 追加）: ceil(月給 × 在籍日数 ÷ 暦日数)。

## 例8: 旧ルール継続・固定月給（フン104、useOldRules=true + salary）— 2026-06-12 改定

**条件:** 固定月給 396,105円・日給 15,693円、6月の全社所定 26日、出勤24日、残業10h、有給0、補償0

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給 | 固定月給（所定日数で変動しない） | 396,105 |
| 残業単価 | ceil(日給 15,693 ÷ 6h40m × 1.25)（固定・月給からの逆算ではない） | 2,943 |
| 残業手当 | ceil(2,943 × 10h) | 29,430 |
| 欠勤控除 | max(0, 26 - 24 - 0 - 0) = 2日 → floor(15,693 × 2) | -31,386 |
| **支給額** | 396,105 + 29,430 - 31,386 | **394,149** |

**ポイント（2026-06-12 是正）:**
- 基本給は**毎月固定 396,105円**。旧実装の「時給 × 所定日数 × 6h40m」は所定日数で月変動するため廃止
- 残業単価・欠勤控除は**日給15,693円ベースで固定**（残業 2,943円/h、欠勤 15,693円/日）
- 所定日数（全社所定 = 日曜以外 − 祝日）は**欠勤日数の算出にのみ**使用
- 月途中退職時は基本給のみ日割り: ceil(396,105 × 在籍日数 ÷ 暦日数)。単価は変わらない
- 有給は月給に内包（取得しても増減なし）

## 監査チェックの不変条件

`PayrollAuditModal` で表示される 4 つの不変条件:

1. **所定労働時間が法定上限以内**: `prescribedHours ≤ legalLimit`
2. **出勤実績の合計が所定日数以内**: `workDays + plDays + restDays + siteOffDays + examDays + compDays ≤ prescribedDays + 1`
3. **支給額の内訳合計が一致**: `sum(各手当) - 欠勤控除 == salaryNetPay` （誤差±1円）
4. **残業倍率が法定下限以上**: `otMul ≥ 1.25`

これらすべて ✓ であれば、支給額は法令準拠で算定されている。
