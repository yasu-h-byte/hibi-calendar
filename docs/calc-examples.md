# 給与計算例集（社労士確認用）

> 各支給項目の計算例と検算ロジックを実例で示します。
> ロジックは `lib/compute.ts:calculateVietnameseSalary` に集約。

## 例1: 標準月（20日勤務・残業なし・補償なし）

**条件:**
- 雇用区分: 時給制ベトナム人（新ルール）
- 時給: ¥1,500、ベース日数: 20、月: 31日（5月）
- 出勤: 20日（各日7h所定通り）
- 有給: 0日、補償: 0日、欠勤: 0日
- 残業: 0h、深夜: 0h、法定休日労働: 0h

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 1,500 × max(0, 20-20) × 7 | 0 |
| 所定外労働手当 | 0h × 1,500 | 0 |
| 法定外残業手当 | 0h × 1,500 × 1.25 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | | **210,000** |

## 例2: 残業発生月（出勤18日 + 有給2日 + 残業18.5h）

**条件:** 例1と同じ条件で、出勤18日、有給2日、残業合計18.5h（うち3層判定 statutoryOT = 1.0h）

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 1,500 × max(0, 18-20) × 7 | 0 |
| **所定外労働手当** | **1,500 × (18.5 - 1.0)** | **26,250** |
| 法定外残業手当 | 1,500 × 1.25 × 1.0 | 1,875 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | max(0, 20 - 18 - 2 - 0) = 0 → -0 | 0 |
| **支給額** | | **238,125** |

**ポイント:** 残業18.5hのうち、3層判定で1.0hだけが法定外残業（1.25倍）に該当。
残り17.5hは「所定外労働手当」として **通常賃金で別途支給**（労基法24条）。

## 例3: 補償日あり（出勤15日 + 補償5日）

**条件:** 例1と同じ条件で、出勤15日、補償（w=0.6）5日

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 0 | 0 |
| 所定外労働手当 | 0 | 0 |
| 法定外残業手当 | 0 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| **休業手当** | **1,500 × 7 × 0.6 × 5** | **31,500** |
| **欠勤控除** | max(0, 20-15-0-0) = 5日 → **-1,500 × 7 × 5** | **-52,500** |
| **支給額** | 210,000 + 31,500 - 52,500 | **189,000** |

**ポイント:** 補償日（休業日）は基本給から1日分を控除（欠勤扱い）、別途 60% を休業手当で支給。
**正味で 60% 支給**（労基法26条準拠）。

## 例4: 法定休日労働（出勤20日 + 日曜10h労働）

**条件:** 例1と同じ条件で、加えて日曜（5/3）に10h労働

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | max(0, 20-20) × 7 × 1,500 | 0 |
| 所定外労働手当 | 通常出勤に残業なしなので 0 | 0 |
| 法定外残業手当 | 法定休日労働は別枠（statutoryOT に含めない） | 0 |
| **法定休日手当** | **1,500 × (1.35 × 8 + 1.60 × 2)** = 1,500 × (10.8 + 3.2) | **21,000** |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | | **231,000** |

**ポイント:** 法定休日（日曜）労働は 8h以下 × 1.35 + 8h超 × 1.60 で算定（労基法37条）。

## 例5: 月60h超残業（出勤22日・残業70h）

**条件:** 例1と同じ条件で、出勤22日、残業合計70h（うち statutoryOT = 70h）

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 1,500 × (22-20) × 7 | 21,000 |
| 所定外労働手当 | 1,500 × (70 - 70) | 0 |
| **法定外残業手当** | **1,500 × (1.25 × 60 + 1.5 × 10)** = 1,500 × (75 + 15) | **135,000** |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | | **366,000** |

**ポイント:** 月60h超の法定外残業は 1.5倍（労基法37条1項ただし書、2023/4 中小企業含む全企業適用）。

## 例6: 深夜労働（出勤20日 + 1日18:00-23:00勤務）

**条件:** 例1と同じ条件で、1日だけ 18:00-23:00 勤務（深夜帯 22:00-23:00 = 1h）

| 項目 | 計算 | 金額 |
|---|---|---|
| 基本給（固定） | 1,500 × 20 × 7 | 210,000 |
| 追加所定 | 0 | 0 |
| 所定外労働手当 | 計算別途必要（実労働超過分） | （計算次第） |
| 法定外残業手当 | 3層判定次第 | （計算次第） |
| 法定休日手当 | 0 | 0 |
| **深夜手当** | **1,500 × 0.25 × 1h** | **375** |
| 休業手当 | 0 | 0 |
| 欠勤控除 | -0 | 0 |
| **支給額** | （基本給 + 各種手当 - 0） | 計算による |

**ポイント:** 深夜手当は他の倍率と独立して +0.25 を加算。
残業中の深夜なら 1.25 + 0.25 = 1.5倍相当。

## 例7: 中途入社月（6/15入社、6月分）

**条件:** 6/15入社（在籍 16/30 暦日 = 53%）、出勤10日（うち全て規定通り）、残業なし

| 項目 | 計算 | 金額 |
|---|---|---|
| 按分後 baseDays | round(20 × 16/30) | 11 |
| 基本給（固定） | 1,500 × 11 × 7 | 115,500 |
| 追加所定 | max(0, 10-11) × 7 × 1,500 | 0 |
| 所定外労働手当 | 0 | 0 |
| 法定外残業手当 | 0 | 0 |
| 法定休日手当 | 0 | 0 |
| 深夜手当 | 0 | 0 |
| 休業手当 | 0 | 0 |
| 欠勤控除 | max(0, 11-10-0-0) = 1日 → -1 × 7 × 1,500 | -10,500 |
| **支給額** | 115,500 - 10,500 | **105,000** |

**ポイント:** 在籍期間で baseDays を按分（暦日数比例）。
中途入社月の過大な欠勤控除を防ぐ。

## 例8: 旧ルール継続者（フン104、useOldRules=true）

**条件:** 6月の月所定 22日、出勤20日、残業10h、有給0、補償0

| 項目 | 計算 | 金額 |
|---|---|---|
| 月所定時間 | 22 × 20/3 | 146.67h |
| 基本給 | 1,500 × 146.67 | 220,000 |
| 残業手当 | 1,500 × 1.25 × 10（全残業 × 1.25） | 18,750 |
| 欠勤控除 | max(0, 22 - 20 - 0 - 0) = 2日 → -1,500 × 20/3 × 2 | -20,000 |
| **支給額** | | **218,750** |

**ポイント:** 旧ルールでは 1日所定 6h40m（20/3）、残業は全時間 × 1.25。

## 監査チェックの不変条件

`PayrollAuditModal` で表示される 4 つの不変条件:

1. **所定労働時間が法定上限以内**: `prescribedHours ≤ legalLimit`
2. **出勤実績の合計が所定日数以内**: `workDays + plDays + restDays + siteOffDays + examDays + compDays ≤ prescribedDays + 1`
3. **支給額の内訳合計が一致**: `sum(各手当) - 欠勤控除 == salaryNetPay` （誤差±1円）
4. **残業倍率が法定下限以上**: `otMul ≥ 1.25`

これらすべて ✓ であれば、支給額は法令準拠で算定されている。
