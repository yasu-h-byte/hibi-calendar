## 日本人社員（日給月給制）

```
基本給 = 日額 × 実出勤日数
残業手当 = (日額 ÷ 8h) × 1.25 × 残業時間
支給額 = 基本給 + 残業手当
```

- 残業単価の分母は **8時間**
- 欠勤控除なし（出勤した日数分だけ支払う仕組み）

## ベトナム人スタッフ（時給制・変形労働時間制）

```
所定時間 = 所定日数 × 7h
基本給 = 時給 × 所定時間
法定残業 = MAX(0, 実労働時間 − 所定時間)
残業手当 = 時給 × 1.25 × 法定残業時間
欠勤控除 = 時給 × 7h × 欠勤日数
欠勤日数 = 所定日数 − 実出勤日数 − 有給日数
支給額 = 基本給 − 欠勤控除 + 残業手当
```

- 残業単価の分母は **7時間**
- 所定時間はカレンダーの所定日数から自動計算
- 有給取得日は欠勤にカウントしない
- 時給制なので現場による所定日数の違いで不公平は発生しない

## 補償（0.6）の扱い
- 外国人の休業補償（w=0.6）: 原価には計上するが、人工数には含めない
- 日本人の0.6: 通常通りカウント（変形労働時間制で基本的に発生しない想定）

## 鳶合計・土工合計のルール
- 鳶合計 = とび + 職長 + 役員 + 外注（鳶）
- 土工合計 = 土工 + 外注（土工）
- 出向者は鳶換算人工（tobiEquiv）から除外するが、出面合計には含む
