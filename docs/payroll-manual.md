## 1. 給料の支払い形態（6パターン）

スタッフは「雇用区分 × 支払い形式 × ルール体系」で 6 パターンに分かれます。

| # | 該当者 | 支払い形式 | ルール | 基本給の決まり方 |
|---|---|---|---|---|
| 1 | **日本人 日給制** | 日給 × 日数 | （変形労働時間制対象外） | 日額 × 出勤日数 |
| 2 | **日本人 月給制** | 月給固定 | （変形労働時間制対象外） | 月給（固定） |
| 3 | **外国人 時給制 新ルール** ※多数派 | 時給 × 時間 | 変形労働時間制 | 時給 × 20日 × 7h |
| 4 | **外国人 月給制 新ルール** | 月給固定 | 変形労働時間制 | 月給（固定） |
| 5 | **外国人 時給制 旧ルール継続** ※現在該当者なし | 時給 × 時間 | 旧ルール（月所定時間） | 時給 × 月所定日数 × 6h40m |
| 6 | **外国人 固定月給 旧ルール継続** ※フン104 | 月給固定 | 旧ルール | **固定月給 396,105円**（所定日数で変動しない）。残業単価2,945円・欠勤控除15,702円/日も固定（日給15,702円ベース） |

### 識別方法

人員マスタ（`/workers`）で確認できます:
- **visa = "none"** → 日本人（パターン1 or 2）
- **visa ≠ "none"** → 外国人（パターン3〜6）
- **salary > 0** → 月給制
- **salary = 0 & hourlyRate > 0** → 時給制（日本人なら日給制 = rate 使用）
- **useOldRules チェック** → 旧ルール継続（個別フラグ）

### 3-3. 給与計算根拠モーダル（クリックで開く）

スタッフ名をクリックすると「🔍 給与計算の根拠」モーダルが開きます。

スクリーンショット配置: `![給与計算根拠モーダル](./images/payroll-audit-modal.png)`

**5セクション構成:**
1. **月情報** - 暦日数、法定上限、所定日数
2. **雇用区分・適用ルール** - 6パターンのどれか、ルール体系、単価
3. **出勤実績の集計** - 出勤・有給・欠勤・残業の内訳（所定外労働と法定外残業を分けて表示）
4. **給与計算（式と結果）** - 各支給項目の計算式と金額
5. **監査チェック** - 4項目すべて ✓ なら法令準拠OK

**監査チェックの4項目:**
- ✓ 所定労働時間が法定上限以内
- ✓ 出勤実績の合計が所定日数以内
- ✓ 支給額の内訳合計が一致
- ✓ 残業倍率が法定下限以上

## 5. よくある質問（FAQ）

### Q1. 出面に残業を入れていないのに、Excelに残業が出ているのはなぜ？

**A.** 3層判定（日8h超／週所定超／月法定上限超）のいずれかが自動判定されています。
- 週の判定基準は「**週所定**」（= MAX(40h, 事前カレンダーで定めた週の所定合計)）です。**事前にカレンダーで6日勤務（週42h）と定めた週は、42hまで残業になりません**（変形労働時間制・2026-06 社労士確認済み）
- カレンダー上の所定を超えて働いた週・日があると、その超過分が法定外残業として自動計上されます

詳細は [calc-examples.md 例2-4](./calc-examples.md) 参照。

### Q2. 補償日 (w=0.6) はどう支払われる？

**A.** 60% 支給（労基法26条準拠）。
- 基本給からは1日分減額（欠勤控除）
- 別途、休業手当 = 時給 × 7h × 0.6 を加算
- 正味: 通常日の 60%
- ⚠️ 2026/6 修正以前は 160% 過払いだった

### Q3. 旧ルール継続スタッフ（フン104）とは？

**A.** 本人希望で新ルール移行を見送ったスタッフ（現在フン104のみ・2027年2月で現雇用契約終了）。
- 人員マスタの「給与計算旧ルール」チェックボックスがON ＋ 固定月給を設定
- **基本給 = 固定月給 396,105円**（出勤日数・所定日数で変動しない。月途中退職は暦日数比で日割り）
- **残業手当 = 2,945円 × 残業時間**（残業単価 = 切上(日給15,702 ÷ 6h40m × 1.25)・固定）
- **欠勤控除 = 15,702円 × 欠勤日数**（日給ベース・固定）
- 退職まで継続。新ルール移行は本人同意が必要

### Q4. 中途入社・退社月の計算は？

**A.** 在籍日数を暦日数で按分。
- 例: 6/15入社（在籍 16/30 = 53%）→ baseDays = round(20 × 53%) = 11
- 基本給も同比率で減額（月給制の場合）
- 欠勤控除も適正に計算される

### Q5. 月60h以上残業した場合は？

**A.** 60h以下は1.25倍、60h超部分は1.5倍（労基法37条1項ただし書）。
- 例: 月70h残業 → 60h×1.25 + 10h×1.5

### Q6. 法定休日（日曜）に長時間労働した場合は？

**A.** 8h以下は1.35倍、8h超部分は1.60倍。
- 例: 日曜10h労働 → 8h×1.35 + 2h×1.60

### Q7. 数字がおかしいときはどう確認する？

**A.** 給与計算根拠モーダルを開き、5つのセクションを上から確認。
1. ① 月情報（法定上限・所定日数が正しいか）
2. ② 雇用区分（パターンが正しいか）
3. ③ 出勤実績（出勤日数・残業時間が出面と一致しているか）
4. ④ 計算式（各項目の式と金額が想定通りか）
5. ⑤ 監査チェック（4項目すべて ✓ か）

❌ が出ている項目があれば、その項目の detail メッセージを確認。

## 6. 給与計算式 早見表

### パターン1: 日本人 日給制（日給月給）

```
基本給   = 日額 × 出勤日数 + 日額 × 0.6 × 補償日数
有給手当 = 日額 × 有給日数                       ← 2026-06 追加（有給日も日給を支給）
残業単価 = 切上(日額 ÷ 8h × 1.25)
残業手当 = 切上(残業単価 × 残業時間)
支給額 = 基本給 + 有給手当 + 残業手当
※ 欠勤控除なし（出た日数分だけ支払う形態のため）
```

### パターン2: 日本人 月給制（完全月給: 政仁・濱上）

```
基本給 = 月給（固定）
       ※月途中入退社のみ日割り: 切上(月給 × 在籍日数 ÷ 暦日数)
残業時給 = 月給 ÷ (月所定日数 × 8h)   ※按分前の月所定で算出（単価は固定）
残業手当 = 切上(切上(残業時給 × 1.25) × 残業時間)
支給額 = 基本給 + 残業手当
※ 欠勤控除なし（出勤日数に関わらず固定）
```

### パターン3-4: 外国人 新ルール（4層構造 + 有給日給）

```
基本給(固定)      = 時給 × 20日 × 7h        ※月給制は固定値
追加所定手当      = 時給 × MAX(0, 出勤日数 − 20) × 7h
有給日給          = 時給 × 7h × 20日枠を超えた有給日数       ← 2026-06 追加
所定外労働手当    = 時給 × 1.0 × 所定外労働h（各日の所定超の全部。法定外残業分の1.0倍もここに含む）
法定外残業手当    = 時給 × 0.25 × 法定外残業（60h超は0.5）   ← 割増分のみ。所定外と合わせ実効1.25倍
法定休日手当      = 時給 × (1.35 × ≤8h + 1.60 × >8h)
深夜手当          = 時給 × 0.25 × 深夜労働h（22:00-5:00）
休業手当          = 時給 × 7h × 0.6 × 補償日数
欠勤控除          = 時給 × 7h × 欠勤日数（補償日含む）

支給額 = 基本給 + 追加所定 + 有給日給 + 所定外労働 + 法定外残業
       + 法定休日 + 深夜 + 休業 − 欠勤控除
```

時給は: hourlyRate（時給制）または salary / (20 × 7)（月給制の逆算）
端数: 支給項目は1円未満切り上げ、控除は切り捨て

### パターン5: 外国人 時給制 旧ルール継続（現在該当者なし）

```
基本給         = 時給 × 月所定日数 × 6h40m
休業補償       = 時給 × 6h40m × 0.6 × 補償日数
残業手当       = 時給 × 1.25 × 全残業時間（3層判定なし）
欠勤控除       = 時給 × 6h40m × 欠勤日数

支給額 = 基本給 + 休業補償 + 残業手当 − 欠勤控除
```

### パターン6: 外国人 固定月給 旧ルール継続（フン104）

```
基本給         = 固定月給 396,105円（出勤日数・所定日数で変動しない）
                 ※月途中入退社のみ日割り: 切上(396,105 × 在籍日数 ÷ 暦日数)
時給換算       = 日給 15,702円 ÷ 6h40m（時給ベースではなく日給ベース・固定）
残業単価       = 切上(時給換算 × 1.25) = 2,945円（固定）
残業手当       = 切上(2,945 × 残業時間)
休業補償       = 切上(日給 × 0.6 × 補償日数)
欠勤控除       = 切捨(日給 15,702 × 欠勤日数)
有給           = 月給に内包（取得しても増減なし）

支給額 = 基本給 + 休業補償 + 残業手当 − 欠勤控除
※ 所定日数（全社所定 = 日曜以外 − 祝日）は欠勤日数の算出にのみ使用
```

## 9. 用語集

| 用語 | 意味 |
|---|---|
| 新ルール | 2026/5〜 1ヶ月単位変形労働時間制 |
| 旧ルール | 〜2026/4 月所定時間ベース |
| ベース日数 | 月給制の基準日数（デフォルト20日） |
| 月所定 | ベース日数 × 1日所定 = 20×7=140h |
| 法定上限 | 暦日数 × 40 ÷ 7（月単位の絶対上限） |
| 法定外残業 | 1.25倍支給対象（3層判定後） |
| 所定外労働 | 月所定超〜法定上限内（通常賃金） |
| 補償日 | 会社都合休業日（w=0.6で入力） |
| 法定休日 | 日曜（1.35倍支給） |
| 所定休日 | 土曜（カレンダーで設定） |
| 3層判定 | 日単位→週単位→月単位の残業認定順 |
| 出向 | 他社へ社員を派遣（dispatchTo で記録） |
