# 労働時間・変形労働時間制

## 所定労働時間
- 1日7時間
- 始業：8:00 / 終業：17:00
- 休憩：10:00〜10:30（30分）+ 12:00〜13:00（60分）+ 15:00〜15:30（30分）= 合計2時間
- 月間所定労働時間：所定日数 × 7h（カレンダーで確定）

## 変形労働時間制（1ヶ月単位）- 2026年5月から適用
- 月の合計で法定上限以内なら、1日8時間を超えても残業にならない
- 残業 = 実労働時間 − 月間所定時間（月の合計で判定）
- 残業割増：25%

## 法定上限早見表

|--------|-------------|-----------------|
| 28日   | 160.0h      | 22日            |
| 29日   | 165.7h      | 23日            |
| 30日   | 171.4h      | 24日            |
| 31日   | 177.1h      | 25日            |

計算式：暦日数 × 40h ÷ 7

## 毎月の運用サイクル
1. 20日頃：元請から翌月カレンダーを入手
2. 25日まで：管理者がシステムに就業カレンダーを入力
3. 月末まで：事業責任者が承認 → 法定上限チェック（自動）
4. 承認後：Messengerでリンク送信 → スタッフがスマホで署名
5. 翌月1日〜：カレンダー通りに勤務、スタッフはスマホで出勤入力
6. 翌月5日頃：月次集計 → 給与計算確認 → 月締め

## カレンダー署名
- 署名後はデータをロックして変更不可
- 署名記録にはタイムスタンプを保存
- 未署名者は通知で表示
