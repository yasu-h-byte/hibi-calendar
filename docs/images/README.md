# スクリーンショット配置先

`docs/payroll-manual.md` で参照される画像をここに配置します。

## 必要な画像

| ファイル名 | 撮影画面 | 撮影タイミング |
|---|---|---|
| `monthly-screen-example.png` | `/monthly?ym=202605` 全体（HFU タブ） | 5月の月次集計を確認時 |
| `payroll-audit-modal.png` | スタッフ名クリックで開く「給与計算根拠」モーダル | サン206 or 同様のスタッフで |
| `monthly-excel-example.png` | 月次集計Excel の HFU 外国人ブロック部分 | Excel出力後にスクショ |
| `hfu-time-sheet.png` | HFU向け勤務時間一覧シート（Sheet 2） | Excel出力後にスクショ |
| `worker-master-edit.png` | 人員マスタ編集モーダル | useOldRules チェックボックスが見える状態 |
| `calendar-approval.png` | カレンダー承認画面 | 承認待ち状態 |

## 撮影のコツ

- PC ブラウザの全画面で撮影（個人情報は黒塗り）
- 1080px幅以上で撮影し、大きすぎる場合は適宜縮小
- ファイル名は半角英数のみ（マニュアル参照を維持）
